Help:

Welcome to the Adventure Editor! Here you can create and edit interactive story adventures.

To get started, you can either load an existing story by clicking the "Load" button and selecting a .zip file, or create a new story by clicking the "New Story" button.

Once you have a story loaded or created, you can add rooms to the story by clicking the "Add Room" button. Each room can have its own text, choices, and outcomes.

When you're ready, you can save your story by clicking the "Save Story" button, which will download a .zip file containing your story files.

'Flowchart' will open in a new tab and should update dynamically as changes are made.

Stories can be uploaded and played via the Main menu.

For more detailed information on file formats and features, please refer to the documentation links provided.
